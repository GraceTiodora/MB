# 🍱 MBGTrack — Sistem Pemantauan Program Makan Bergizi Gratis

Platform digital untuk monitoring distribusi program makan bergizi (MBG) nasional dengan dashboard analitik, peta interaktif, dan sistem pelaporan publik.

## 📊 Fitur Utama

### Landing Page
- Hero section dengan background carousel (7 gambar)
- KPI snapshot operasional real-time
- Distribusi per wilayah preview
- Kolaborasi lintas kementerian (marquee animation)
- Status laporan terbaru dengan galeri
- Call-to-action terstruktur

### Dashboard
- KPI cards dengan animasi
- Distribusi per provinsi (Top 10)
- Alert panel dengan status priority
- Chart tren distribusi mingguan
- Quick access navigation

### Peta Wilayah (Region)
- Interactive map Leaflet.js dengan 7 provinsi
- Markers dengan status color-coded
- Detail popup saat klik marker
- Search & filter region
- Progress tracking per sekolah

### Tracker Distribusi
- Status pengiriman real-time
- Timeline tracking untuk shipment
- Search by tracking ID
- Alert untuk masalah pengiriman

### Laporan Publik
- Galeri laporan lapangan (4 foto)
- Filter by status
- Comment system
- Real-time update

### Nutrition & Feedback
- Report kesehatan & gizi
- Form laporan masalah dengan geolocation
- Tracking laporan status

### Admin Panel
- User management
- Report moderation
- Data validation
